forms are created useing the 'form' element. The form accepts two essential attributes; the 'action' attribute takes a URL value that tells the form where to send data to. The 'method' attribute tells the browser which http request method is should use to submit the form. (GET, POST request methods) GET retrieves data from the server, while POST changes something in the server. use form control elements to collect user data. e.g. text boxes, dropdowns, checkboxes, buttons.
